# EmpNOW

**EmpNOW (Empirical Non-Overlapping Windows)** is an R pipeline to perform non-overlapping windows analysis in empirical dataset.

Software incorporated in the pipeline:
- <a href="http://www.iqtree.org">IQ-TREE 2</a>
